# Data Access & Preprocessing

This folder contains instructions and helper scripts for obtaining and preprocessing
data required by VisioGen.

## Recommended Data Sources

- **GWAS Catalog**: Download summary statistics for eye disease traits.
- **UK Biobank**: Access individual-level genotype and eye phenotype data.
- **Retinal imaging datasets**: e.g., REFUGE / ORIGA for labeled fundus photographs.

## Notes

- Many datasets require controlled access and data use agreements.
- Store downloaded data outside of version control (e.g., in a `data/` directory that is gitignored).
